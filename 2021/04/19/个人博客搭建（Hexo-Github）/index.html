个人博客搭建（Hexo + Github）

个人博客搭建（Hexo + Github）

(Windows 10环境下)

1、下载安装git，一路默认就行，完成后会有 Git Bash 、Git GUI等，要使用的是Git Bash，方便操作（官网）

2、下载安装nodejs，因为Hexo是基于Nodejs写的，所以需要安装nodejs和npm工具，也一路默认（官网）

​ Windows安装LTS版本

​ 上面两步完成后，打开Git，输入 git –version node -v npm -v 即可查看已版本，判断是否安装成功

3、在Git输入命令 npm install -g hexo-cli ，即可安装Hexo，然后输入 hexo -v 即可查看版本

​ （也有安装cnpm即npm镜像版本，速度更快，没试过）

​ npm install -g cnpm –registry=http://registry.npm.taobao.org # 安装淘宝的cnpm管理器
​ cnpm -v
​ cnpm install -g hexo-cli

4、新建空文件夹 blog，右击选择 Git Bash Here，然后输入 hexo init # 初始化Hexo

​ （或通过命令行创建 mkdir 新的文件夹并进入 cd blog）

​ （文件夹一定要建在 C:\Users\24545\ 目录下，否则用Gitbash输入接下来的命令会报错，即没有权限，）

​ （3、4步先后顺序无所谓，好像最后npm、hexo文件都在同一个文件目录下）

5、输入 npm install (这步不要好像也行！？)

​ hexo s/hexo server # 启动本地博客服务

​ （在浏览器输入 http://localhost:4000 即可访问，ctrl + C 可关闭）

6、输入 hexo n “Hello Hexo” # 创建新博客

​ hexo clean # 清理

​ hexo g/hexo generate # 生成

​ hexo s

​ 然后刷新一下 localhost:4000，即可看到新写的博客

7、将本地博客与github关联，在github上新建一个仓库（New repository），名字与用户名相同，尾部加 .github.io

点击 Create repository ，此时仓库为空，什么都没有

​ （一定要设为公开的 public ，设为私有的 private 会看不到）

8、回到 Gitbash 中，输入 git config –global user.name “YOURNAME”

​ git config –global user.email “YOUREMAIL”

​ （设置 Git 账户，名字和邮箱均为github的用户名和邮箱）

​ 输入 git config user.name/user.email 即可查看

​ 然后输入 ssh-key -t ras -C “YOUREMAIL” ，创建SSH，将SSH添加到Github，一路默认即可

​ 完成后会在目录下生成 .ssh 文件夹，里面的 id_rsa.pub 为公共密钥，另一个 is_rsa 为私有密钥

​ 用记事本打开 id_rsa.pub 并复制，然后打开 github 账户的设置(Settings)，

​ 选择 SSH and GPG keys，点击 New SSH key ，title 随便取，key 即粘贴刚刚复制的公共密钥

​ 再回到 Gitbash ，输入 ssh -T git@github,com ，可查看是否安装成功

​

9、打开 blog 目录下的 _config.yml 文件（记事本），拉到末尾，将 deploy 改为

​ deploy:
type: git
repo: https://github.com/yourgithubname/yourgithubname.github.io.git # 注意改
branch: master

​ （注意格式，type、repo、branch前面均有两个空格，“:”后均有一个空格）

​ 然后输入 npm install hexo-deployer-git –save

​ 再输入 hexo clean hexo g hexo d/hexo deploy

​ 输入hexo d 后，要输入 github 账户的密码，一次没出现在输入一次就行（我就是，第一遍没跳要输入密码）

​ 最后在自己新建的仓库刷新一下就有了，然后就可以通过 https://yourgithubname.github.io/ 访问了
